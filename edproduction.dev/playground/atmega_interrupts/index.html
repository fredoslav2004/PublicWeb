01. The Mechanism

Polling is deterministic but wasteful. Interrupts are non-deterministic (temporally) but efficient. On the AVR architecture, an interrupt is a hardware signal that forces the CPU to pause normal execution.

The Sequence:

  1. Event sets a flag bit (e.g., INTF0).
  2. If I-bit in SREG is 1 and mask is enabled:
  3. CPU finishes current instruction.
  4. PC (Program Counter) is pushed to Stack.
  5. I-bit is cleared (interrupts disabled).
  6. CPU jumps to Vector Address (e.g., 0x0002 for INT0).
  7. ISR executes.
  8. RETI instruction pops PC and re-enables interrupts.

03. Register Configuration

The ATmega2560 requires specific bit sequences to enable external interrupts. Unlike attachInterrupt(), understanding these registers allows you to optimize code size and latency.

04. Bare Metal Implementation

External Interrupt (INT0 on PD0) 
#include <avr/io.h>
#include <avr/interrupt.h>

// Volatile is mandatory. Compiler assumes variable unused otherwise.
volatile uint8_t state = 0; 

void setup_interrupts() {
    // 1. Configure PD0 (INT0 pin) as input
    DDRD &= ~(1 << DDD0); 
    PORTD |= (1 << PORTD0); // Enable pull-up

    // 2. Configure EICRA (External Interrupt Control Register A)
    // ISC01=1, ISC00=0 -> Falling Edge
    // Table 15-2 in Datasheet
    EICRA |= (1 << ISC01);
    EICRA &= ~(1 << ISC00);

    // 3. Enable INT0 in Mask Register
    EIMSK |= (1 << INT0);

    // 4. Enable Global Interrupts (sets I-bit in SREG)
    sei(); 
}

// Vector Name found in 
ISR(INT0_vect) {
    state = !state;
    // Keep ISRs SHORT. No Serial.print here.
}

int main(void) {
    setup_interrupts();
    
    // Configure built-in LED as output
    DDRB |= (1 << DDB7);

    while(1) {
        if(state) {
            PORTB |= (1 << PORTB7);
        } else {
            PORTB &= ~(1 << PORTB7);
        }
    }
}
Timer1 Compare Match (CTC Mode) 
// Generate an interrupt exactly every 1 second (1Hz) at 16MHz
void setup_timer1() {
    // Disable global interrupts during setup
    cli();

    // Reset control registers
    TCCR1A = 0;
    TCCR1B = 0;

    // Set Compare Match Register value
    // (16*10^6) / (1024 * 1) - 1 = 15624
    OCR1A = 15624;

    // Turn on CTC mode (Clear Timer on Compare Match)
    // WGM12 bit in TCCR1B
    TCCR1B |= (1 << WGM12);

    // Set Prescaler to 1024
    // CS12 and CS10 bits
    TCCR1B |= (1 << CS12) | (1 << CS10);

    // Enable Timer Compare Interrupt
    TIMSK1 |= (1 << OCIE1A);

    // Enable global interrupts
    sei();
}

ISR(TIMER1_COMPA_vect) {
    // Toggle LED every second hardware-precisely
    PORTB ^= (1 << PORTB7);
}

05. Critical Hazards

The Volatile Keyword

The compiler optimizes code assuming sequential execution. If a variable is modified in an ISR but only read in main(), the compiler may cache the value in a register and never reload it from RAM.

// WRONG
int flag = 0;

// RIGHT
volatile int flag = 0;

Atomic Access

ATmega2560 is an 8-bit architecture. Reading/Writing a 16-bit integer (like int) takes two clock cycles. An interrupt can occur between the low byte and high byte operations.

uint16_t val;

// Atomic Block
cli(); // Disable Interrupts
val = sharedVar; // Safe Read
sei(); // Re-enable
