Types of Functions

  1. Injective (One-to-One): No two elements in A map to the same element in B.
  2. Surjective (Onto): Every element in B is mapped to by at least one element in A.
  3. Bijective: Both injective and surjective (invertible).

Coding: Higher-Order Functions

In programming, functions are first-class citizens.

const numbers = [1, 2, 3, 4];
